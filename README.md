# video-editors

Ascendante — AI video ad generation and editor (Next.js, Inngest, Remotion/OpenVideo).

## Setup

1. Copy `.env.example` to `.env` and fill in API keys (never commit `.env`).
2. `npm install`
3. Run Inngest dev and the app: `npm run dev` (port 4000).

## Notes

- Secrets must be provided via environment variables (`PHONOS_TOKEN`, `CAMB_API_KEY`, etc.).
- Sync upstream from [ascendantecommerce/maker](https://github.com/ascendantecommerce/maker) via `origin`; client backup at [MuhammadMuazzain/ascendanteecommerce](https://github.com/MuhammadMuazzain/ascendanteecommerce).
